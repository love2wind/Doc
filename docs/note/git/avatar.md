### 搭建方法

本项目就是html+js，支持静态网页的平台都可以直接搭建使用，我这里使用了vercel.com，所以你只要fork项目，修改 `index.html`文件的标题和相关文本，然后修改 `config.json` 里配置就可以直接部署了。如果你有自己的服务器，就直接新建网站，下载项目代码，修改文件后直接就可以访问了。
