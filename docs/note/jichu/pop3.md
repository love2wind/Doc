## Outlook设置

#### POP3设置

```
服务器名称: outlook.office365.com
端口: 995
加密方法: TLS
```

#### IMAP设置

```
服务器名称: outlook.office365.com
端口: 993
加密方法: TLS
```

#### SMTP设置

```
服务器名称: smtp.office365.com
端口: 587
加密方法: STARTTLS
```
